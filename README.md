# WebPack4

## git 命令

`git init` 初始化本地git仓库
`git add remote <远程仓库名称> <远程仓库url>` 添加远程仓库
`git remote` 查看本地关联的远程仓库
`git remote set-url <远程仓库名称> <远程仓库地址>` 修改远程仓库地址         
`git push -u <远程仓库名称> <本地分支>` 将本地分支上传到远程合并，并记录push到远程分支的默认 下次直接 git push
